Mp3 Music Player

Do you like music? Then give it a try!

A simple but full-featured Desktop MP3 Player!

This is the mp3 player we all used as kids. Once you move a file by drag and drop, it will automatically change to the absolute path to add it to the list, and you can easily add and delete lists with the Add and Remove buttons. There is a playlist function, and music is generated when you play the added mp3 file. When the music is played, you can see the music being played and adjust the music duration with the slider. The added music lists are managed in a database, so the lists are saved and remain even after the program is closed. Play, Pause, Stop, Prev, and Forward functions are available. I used Java (JavaFX) to implement the program functions, GUI, and database connection, CSS for color and design, and Sqlite to implement the database.

Language Used: Java(JavaFX), CSS, SQLite.

Password Checker

You must check your password frequently. If your passwords are leaked when you check with this program. You should change them right now!

Using Python 3, this program leverages the hashlib library to securely check if a given password has been previously compromised. By employing the SHA-1 hashing algorithm, user passwords are converted to SHA-1 hexadecimal format, ensuring data integrity and confidentiality. Utilizing the Have I Been Pwned API, the program checks if the hashed password matches any known compromised passwords. This approach allows users to verify their passwords without encountering barriers, providing an added layer of security in password management.

Language Used: Python3
